Understanding the usermod Command

Basic Usage of usermod

The usermod command is used to modify an existing user account on a Linux system. It allows administrators to change various user account attributes and settings.

usermod [OPTIONS] USERNAME

Common Options for usermod

The usermod command supports a variety of options:

  • -d, --home - Change the user's home directory.
  • -m, --move-home - Move the contents of the user's home directory to the new location specified by -d.
  • -s, --shell - Change the user's login shell.
  • -G, --groups - Specify a comma-separated list of supplementary groups for the user.
  • -u, --uid - Change the numerical user ID for the user.
  • -g, --gid - Change the user's primary group.
  • -l, --login - Change the user's login name.
  • -p, --password - Change the user's password (encrypted).
  • -e, --expiredate - Set an expiration date for the user's account.
  • -f, --inactive - Set the number of days after a password expires until the account is disabled.
  • -h, --help - Display help information and exit.

Examples of usermod Command

Change User's Home Directory

sudo usermod -d /new/home/dir username

This command changes the home directory of username to /new/home/dir.

Move User's Home Directory

sudo usermod -m -d /new/home/dir username

This command changes the home directory of username to /new/home/dir and moves the contents from the old directory to the new one.

Change User's Shell

sudo usermod -s /bin/zsh username

This command changes the login shell of username to Zsh.

Add User to a Group

sudo usermod -aG groupname username

This command adds username to the supplementary group groupname without removing them from other groups.

Change User's Login Name

sudo usermod -l newusername oldusername

This command changes the login name of oldusername to newusername.

Understanding User Modification Process

When a user account is modified using the usermod command:

  • The user's information is updated in the /etc/passwd file.
  • If the home directory is changed, the contents may be moved to the new location if -m is specified.
  • Group memberships may be altered based on the specified options.
